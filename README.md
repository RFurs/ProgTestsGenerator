# ProgTestsGenerator
ProgTestsGenerator – tai yra Java programa skirta tam tikro kognityvinio lygio pagal Blumo taksonomiją programavimo testų generavimui naudojant Gemini arba ChatGPT DKM ir leidžianti patikrinti atsakymų teisingumą bei atitikimą Blumo taksonomijai pasinaudojant "LLM-as-a-Judge" sistema.

# Programos paleidimas
1) Atsisiųsti programą
2) Atidaryti pom.xml failą mėgstamiausioje IDE. Turi būti galimybė atidaryti kaip projektą
3) Sukurti ir nukopijuoti Gemini bei ChatGPT API raktus
4) Paleisti Application.java failą
5) Įvesti anksčiau sukurtus raktus, papildomai galima kitame skirtuke pasirinkti DKM modelius
6) Galima generuoti testus
